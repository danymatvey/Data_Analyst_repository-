Пришел в аналитику данных, сменив карьерное направление, поскольку эта сфера сочетает в себе логику, творческий подход и возможность извлекать практическую пользу из информации. Мой опыт на различных должностях показал, что я неплохо нахожу точки роста и оптимизирую процессы с помощью анализа, что неоднократно положительно оценивалось руководством. Мне нравится выстраивать ясную картину из разных данных, находить закономерности и влиять на эффективность бизнеса.

# Описания проектов:
 # - Агентство недвижимости;
 Стек: PostgreSQL, Datalense
 Выводы: определил популярные сегменты (2-комнатные квартиры), сезонные пики активности и перспективные города-спутники с высокой долей продаж
 # - Гипотезы и А/В-тесты;
 Стек: Python (Pandas, matplotlib, seaborn, NymPy, statmodels), А/В-тесты, статистика
 Выводы: подтвердил статистическую значимость улучшения ключевой метрики (доли успешных сессий) и рекомендовал внедрить новый алгоритм
 # - Изучение рынка заведений общественного питания Москвы;
 Стек: Python (Pandas, matplotlib, seaborn, NymPy, statmodels)
 Выводы: определил лидирующие категории (рестораны, кафе), выявил доминирование ЦАО и обнаружил, что сетевые кофейни и пиццерии имеют высокие рейтинги;
 # - Исследование стартапов;
 Стек: Python (Pandas, matplotlib, seaborn, NymPy, statmodels)
 Выводы: выявил доминирование IT-сферы и США, снижение типичного размера финансирования и определил наиболее успешные категории стартапов (недвижимость, биотех);
 # - Секреты Темнолесья;
 Стек: PostgreSQL
 Выводы: выявил, что 18% игроков платят за внутриигровую валюту, при этом ключевые игровые предметы покупаются массово, а выбор расы влияет на частоту и сумму трат;
 # - Сервис доставки еды;
 Стек: PostgreSQL, метрики, Datalense
 Выводы:  Выявил низкое удержание пользователей и проблемы со стабильностью приложения, предложив меры по стимулированию активности и улучшению качества;
 # - Яндекс Афиша;
 Стек: PostgreSQL, Python (Pandas, matplotlib, seaborn, NymPy, statmodels), А/В-тесты, статистика, метрики, Datalense
 Выводы: выявил сезонные тренды, пики спроса из-за акций, лидеров по выручке и подтвердил гипотезы о более высокой активности пользователей мобильных устройств.
